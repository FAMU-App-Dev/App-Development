# App-Development

1. User Stories

  Required:
    -User login 
    -User recommendations for each category (movies, tv shows, music, etc.)
    -Description of why it was recommended
    -Profile Pages(name, bio, reommendations etc.)
    -Friends List
    -Search Bar (places, friends, specific movies)
    -Settings (Accessibility, Notification, General etc.)
    
  Nice-to-Haves:
    -Explore page
    -Saved recommendations
    -Recommendation likes
    -Chat 
    
2. Screens

  -Login Screen
  -Register Account Screen
  -Profile Page Screen
  -Friends List Screen
  -Settings Screen
  -Search Bar Screen
  -Chat Screen
		
  
  
3. Tab Navigation:
 
  -Friends
  -Home
  -Chat
  -User Profile
